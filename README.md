# The 50 Problem Recursion Gauntlet

Este repositório contém a resolução progressiva de uma lista de **50 problemas focados em recursão**, desenvolvidos como forma de **treino, aprofundamento lógico e consolidação do conceito de funções recursivas em C++**.

Este é um **repositório em constante atualização**: novos exercícios são adicionados conforme são resolvidos.

---

## Objetivo do repositório

- Praticar recursão de forma intensiva
- Desenvolver raciocínio lógico aplicado a problemas recursivos
- Entender casos base e chamadas recursivas
- Aprimorar a leitura e escrita de código em C++

Este repositório funciona como um **diário de treino**, e não como um projeto finalizado.

---

## Conteúdos abordados

Os exercícios envolvem conceitos como:

- Casos base e chamadas recursivas
- Recursão simples e múltipla
- Problemas matemáticos
- Manipulação de vetores e strings com recursão
- Análise de chamadas recursivas
- Treino de lógica aplicada
